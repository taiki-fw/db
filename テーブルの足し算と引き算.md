# 集合演算

レコード同士の四則演算

## UNION

重複行を排除し、テーブルの合成

```sql
SELECT shohin_id, shohin_mei
FROM Shohincopy
UNION
SELECT shohin_id, shohin_mei
FROM Shohin2 ORDER BY shohin_id ASC;
```

## INTERSECT

テーブルの共通部分の選択

```sql
SELECT shohin_id, shohin_mei
FROM Shohincopy
INTERSECT
SELECT shohin_id, shohin_mei
FROM Shohin2 ORDER BY shohin_id ASC;
```
